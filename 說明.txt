Kuro 專案啟動說明
=================

這份是手動啟動各服務時用的備忘錄。  
以下範例都改成以「repo 根目錄」為前提，不再綁死固定磁碟路徑。

先切到 repo 根目錄：

```powershell
cd <repo-root>
$RepoRoot = (Get-Location).Path
```

GPU 環境變數
------------

```powershell
$env:CUDA_VISIBLE_DEVICES = "0"
$env:PYTORCH_CUDA_ALLOC_CONF = "expandable_segments:True"
```

TTS
---

### GPT-SoVITS WebUI

```powershell
Set-Location (Join-Path $RepoRoot "gpt_sovits")
.\.venv\Scripts\Activate.ps1
python .\webui.py
```

### GPT-SoVITS API v2

```powershell
conda activate (Join-Path $RepoRoot "envs\kuro-tts310")
Set-Location (Join-Path $RepoRoot "gpt_sovits")
python .\api_v2.py -a 127.0.0.1 -p 9881
```

LLM
---

```powershell
conda activate (Join-Path $RepoRoot "envs\kuro-llm310")
Set-Location (Join-Path $RepoRoot "Open-LLM-VTuber")
$env:OPEN_LLM_VTUBER_CONFIG = (Join-Path $RepoRoot "Open-LLM-VTuber\conf.yaml")
python .\run_server.py
```

Bridge / 日文 spoken 轉換
------------------------

```powershell
$env:OPENAI_API_KEY = "<your-openai-key>"
$env:OPENAI_TRANSLATE_MODEL = "gpt-5-mini"
$env:KURO_OPEN_LLM_DIR = (Join-Path $RepoRoot "Open-LLM-VTuber")
$env:KURO_MODEL_DICT_PATH = (Join-Path $RepoRoot "Open-LLM-VTuber\model_dict.json")

Set-Location (Join-Path $RepoRoot "bridges")
python -m uvicorn deeplx_bridge:app --host 127.0.0.1 --port 1188
```

Docker 指令
-----------

```powershell
docker compose ps
docker compose logs -f deeplx
docker compose restart deeplx
docker compose stop deeplx
docker compose rm -f deeplx
docker compose pull deeplx
docker compose up -d deeplx
```

Launcher
--------

```powershell
cd <repo-root>
.\envs\kuro-llm310\python.exe .\launcher.py
```

如果要用根目錄的 VBS 啟動器，現在也會以該檔所在資料夾當成 repo root。
